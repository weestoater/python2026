## Quick Start

1. **Clone this repository**:
   ```bash
   git clone https://github.com/weestoater/python2026.git
   cd python2026
   ```

2. **Make sure Python is installed**:
   ```bash
   python --version
   # or
   python3 --version
   ```
   You should see Python 3.8 or higher.

3. **Run the tutorial programs**:
   ```bash
   # Start with Hello World
   python hello_world.py
   
   # Learn about data structures
   python data_structures.py
   
   # Explore functions and classes
   python functions_and_classes.py
   ```

## What's Included

- **hello_world.py** - Your first Python program
- **data_structures.py** - Learn about lists, tuples, dictionaries, and sets
- **functions_and_classes.py** - Master functions and object-oriented programming
